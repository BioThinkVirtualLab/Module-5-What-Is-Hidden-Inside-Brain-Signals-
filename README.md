# BioThink NeuroLab — Module 5, Lab 07

## Hidden Patterns in Brain Signals

This interactive simulation introduces:

- EEG synchrony and signal amplitude
- Slow and fast brain rhythms
- Motor imagery and rhythm suppression
- Event-related potentials and trial averaging
- AI comparison of EEG signal features

## Publish with GitHub Pages

1. Upload `index.html`, `README.md`, and `.nojekyll` to the root of the repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, select:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /(root)
4. Click **Save**.

Your URL will normally be:

`https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/`
